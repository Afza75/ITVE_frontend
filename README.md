**Project Overview**
This project is a redesigned landing page for the Institutes of Technical & Vocational Education (ITVE), created as part of the Frontend Internship Assessment.
The aim was to build a modern, professional, and responsive landing page that clearly presents ITVE, its programs, and encourages students to take action.

**Tech Stack**
- HTML5
- CSS3
- JavaScript

**How to Run Locally**
- Download or clone the repository
- Open the project folder
- Open index.html in any modern web browser

**Pagination Approach**
Section-based pagination is implemented using dot navigation and smooth scrolling.
Each dot represents a section of the landing page. As the user scrolls, the active section is detected using JavaScript and the corresponding dot is highlighted.
Users can also click on any dot to directly navigate to a specific section.

**Key Design Decisions**
- Used a single consistent color theme for a professional look
- Designed full-screen sections for clear content separation
- Added clear call-to-action buttons
- Ensured full responsiveness for mobile, tablet, and desktop
